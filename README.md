# 🦊 Chrome Extension Pet

> VSCode의 펫과 유사한 기능을 가진 크롬 확장 프로그램

## 📋 목적
VSCode의 펫과 유사한 기능을 가진 기능 개발

## 🚀 Version 0.0.1 

### ✨ 기능개발
- **마우스 클릭** → 마우스클릭 위치 따라감
- **마우스 더블클릭** → 마우스클릭 위치에 따라 뛰어감 
- **펫 드래그** → body에서 드래그된 곳으로 옮겨짐

### 🐛 버그사항
- 펫 드래그 후 (body가 아닌 곳에서) 더블클릭할 경우 부모 태그를 벗어나는 현상이 간혹 발생
